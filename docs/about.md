# <ruby>关于本站 <rt>About This Site</rt></ruby>

> “博观而约取，厚积而薄发。” —— 构建这个站点的初衷，是为了给飞逝的时光留下可以触摸的痕迹。



## 🖋️ 缘起：为何选择 Markdown？

在碎片化信息爆炸的时代，**Markdown** 是我最后的坚持。

对于开发者而言，Markdown 不仅仅是一种标记语言，它更是一种**“心流”的载体**。它让我能够：
- **专注内容**：无需在排版工具的按钮间迷失，只需按下 <kbd>Ctrl</kbd> + <kbd>S</kbd>，思想即成文章。
- **极客审美**：纯文本的底色，配合精准的代码高亮，这是属于程序员的浪漫。
- **持久化生命力**：它是通用的、开放的，即使十年后，这些文字依然清晰可见，不被任何平台裹挟。



## 🚀 始于当下：创建本站的初衷

站名取为 <mark>不负年华</mark>，寓意在最好的年纪，做最深度的思考。

1. **对抗遗忘**：好记性不如烂笔头。通过系统化输出，将“学过的知识”转化为“长在脑子里的能力”。
2. **深度耕耘**：在 VitePress 的纯净环境下，拒绝浮躁，沉下心来打磨每一篇笔记。
3. **审美实践**：将 JetBrains 的设计美学融入 Web，打造一个既好用又好看的“数字花园”。



## 🛠️ 极客底座：技术栈与部署

本站并非一个简单的模板，而是经过深度定制的“个人生产力终端”。

### 技术架构
- **核心框架**：[VitePress](https://vitepress.dev/) <Badge type="tip" text="v1.x" /> —— 极速的开发体验，极致的阅读感受。
- **设计语言**：**JetBrains Style**。我们手动注入了 JB 标志性的 `橙-红-紫` 渐变色。
- **动态特性**：
  - **全自动侧边栏**：基于 `sidebar-auto.js` 脚本，实现文件变动即刻同步，无需手动维护。
  - **沉浸式交互**：定制化极客光标 `<kbd>` 样式，以及偶尔飘落的秋叶动效。
